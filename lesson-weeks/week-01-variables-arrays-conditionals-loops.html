JavaScript Functions and Parameters

Finally, we have the function. Here we use a function called "toCelsius" that accepts the parameter "fahrenheit" from its caller. The caller sends the number 77 as the "fahrenheit" parameter to the function. The function takes the number and returns the result of an equation which made use of the parameter 77. The return value is the 77 Fahrenheit temperature converted to 25 Celsius temperature.
