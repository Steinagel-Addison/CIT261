Storing Simple Data

By using this web storage API, we can store simple key/value pairs of data (strings, numbers, etc) and retrieve these values when needed.

Getters and Setters

Here I have used "localStorage.setItem('firstName', 'Addison')" in order to store a key called 'firstName' with the value "Addison".

I then used "var myName = localStorage.getItem('firstName')" in order to assign the value retrieved by getItem() from the item 'firstName' and assign the value to 'myName'.

Deleting items

We can also remove that item out of web storage by calling the "removeItem()" method.

We can also remove everything in the array from the object with "clear()".

Finding an item or getting the length

Finally, we can find a specific item in the object with "key(n)", or the number of items in the object with "length". The array is alphabetically sorted by key.
